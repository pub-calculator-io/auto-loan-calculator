This free loan calculator computes the monthly payment and overall cost of an auto loan while taking into consideration sales tax, costs, trade-in value, and other factors.

[https://www.calculator.io/auto-loan-calculator/](https://www.calculator.io/auto-loan-calculator/)

== Usage ==

Add the Auto Loan Calculator shortcode to your page, post or sidebar:

`[ci_auto_loan_calculator]`

Add the following code to your template where you would like the Auto Loan Calculator to appear:

`<?php display_ci_auto_loan_calculator(); ?>`

2. Activate the Auto Loan Calculator plugin through the "Plugins" menu in WordPress.
